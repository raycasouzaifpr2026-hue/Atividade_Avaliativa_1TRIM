Elementos html: h1 ao h6

Explicando o h1, h2, h3, h4, h5 e h6

O que são os elementos h1 ao h6?

São conhecidos como Heading Tags (Tags de Cabeçalho). No HTML, eles servem para definir os títulos e subtítulos de uma página, criando uma estrutura organizada para quem lê.

Como usamos?

Para usar os elementos de <h1> a <h6> na prática, você deve inseri-los dentro do corpo, <body> do seu arquivo HTML, envolvendo o texto que você deseja transformar em título.

Por exemplo:

Este é o título principal

Este é um subtítulo importante

Pense nesses elementos como um livro, onde o h1 é o título, h2 os capítulos e h3 os subcapítulos. É como uma hierarquia

O elemento <h1> seria o como o título principal da página

<h2> as seções principais do conteúdo

<h3> subseções que detalham um <h2>

E do <h4> a <h6> seriam usados em níves de detalhamento mais profundos, embora raramente chegue no <h6>

Exemplo:

<h1>Guia Completo de Culinária Italiana</h1>

<h2>Massas Frescas</h2>

<h3>Como escolher a farinha</h3>

<h3>Tempo de cozimento ideal</h3>
